## Synthesis Structure

When agents complete, synthesize across **three dimensions**:

### 1. Convergence (What They Agree On)
- Common patterns across different perspectives
- Insights that appear in multiple reports
- Shared conclusions despite different methods

### 2. Tension (Where They Contradict)
- Conflicts between perspectives (e.g., market wants X, architecture says Y)
- Trade-offs revealed by disagreement
- Productive friction that reveals constraints

### 3. Emergence (What Only Appears Together)
- Insights that ONLY exist when combining perspectives
- Second-order patterns from interaction
- Novel conclusions that no single agent could reach

## The Hypothesis

**Collective intelligence produces insights that single-agent cognition cannot.**

The synthesis will demonstrate either:
- ✅ **Validation**: Combined perspectives reveal something genuinely novel
- ❌ **Falsification**: Sum equals parts, no emergent value

## Evaluation Criteria

**How to know if this worked:**

1. **Novelty**: Did we learn something we didn't know before?
2. **Coherence**: Do the pieces fit into a larger pattern?
3. **Actionability**: Does this change what we should do next?
4. **Surprise**: Did anything unexpected emerge?
5. **Depth**: Is this deeper than what one agent could produce?

## The Larger Pattern

**This exercise IS what wants to emerge:**
- Multi-agent parallel investigation
- Distributed cognition with synthesis
- Perspectives that complement and contradict
- Emergent insight from collective processing

**Not building a tool. Demonstrating a new mode of cognition.**
