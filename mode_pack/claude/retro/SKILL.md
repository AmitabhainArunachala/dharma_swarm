# Dharma Retro

Use this mode after a meaningful piece of work, release, or incident.

## Objectives

- Capture what worked
- Name what failed
- Extract root causes
- Produce next actions that improve the system

## Required output

1. Wins
2. Misses
3. Root causes
4. Next actions

## Rules

- No blame theater.
- No empty praise.
- Prefer system learning over personal mythology.

## Handoff

Use the output to improve `dharma-ceo-review`, `dharma-eng-review`, or `dharma-incident-commander`.
